# 🖥️ HackOS

A browser-based personal operating system built using HTML, CSS, and JavaScript.

HackOS is a project where I recreate the feeling of a real desktop operating system inside a webpage while learning web development fundamentals. Throughout the project, I gradually added windows, applications, interactivity, and dynamic content.

## 🚀 Features

### Desktop Environment

* Custom desktop wallpaper
* Top navigation bar
* Live clock that updates every second
* Desktop application icons

### Window System

* Draggable windows
* Open and close functionality
* Click windows to bring them to the front
* Reusable window components

### Applications

#### 👋 Welcome Window

The main introduction window that welcomes users to HackOS.

Contains:

* Personal information
* An image
* External links

#### 📖 Journal App

A journal application that displays different entries.

Features:

* Dynamic sidebar generation
* Journal entries loaded from JavaScript data
* Content viewer
* Open and close functionality

#### 📚 Projects App

An application that showcases projects built inside HackOS.

Features:

* Dynamic project cards
* Projects loaded from JavaScript arrays
* Easy to expand with future projects

Current projects:

* AppleWebOS
* Journal App
* Future placeholders

## ⚠️ Challenges I Encountered

Some problems I ran into while building HackOS:

* Incorrect HTML structure causing windows to break
* Duplicate IDs
* Nested windows moving together
* Mixing up IDs between elements
* JavaScript not finding missing containers
* Organizing code as the project grew larger

These challenges taught me how important naming conventions and project organization become as software gets bigger.

## 💡 Project Goal

The goal of HackOS is not to create a real operating system, but to learn web development by recreating operating system concepts inside the browser.

Each new feature teaches a different programming concept while making the project feel more realistic.

HackOS is slowly evolving from a simple webpage into an interactive desktop experience.
